# 창원시 컨설팅 워크스페이스 첫 사용 안내

접속 주소: https://consulting.jigooo.com

## 1. 처음 들어오기

1. 위 주소로 접속합니다.
2. 회원가입을 누릅니다.
3. 이름, 이메일, 비밀번호를 입력합니다.
4. 가입 후 자동으로 개인 워크스페이스가 만들어집니다.

주의: 현재는 공개 회원가입 방식입니다. 주소를 받은 사람은 가입할 수 있습니다. 대신 과도한 요청은 자동 제한됩니다.

## 2. 창원시 프로젝트로 일하는 방식

권장 구조는 아래와 같습니다.

- 프로젝트: 창원시 컨설팅
- 채널: 자료수집, 분석, 보고서, 회의록, 질의응답
- 토픽: 각 채널 안의 세부 주제
- 스레드: 실제 대화와 산출물 단위

예시:

- 자료수집 > 공공시설 기초자료 > 2026년 1차 자료 확인
- 분석 > 시설 적정성 진단 > 선별지표 점검
- 보고서 > 중간보고서 > 1차 초안 작성

## 3. 지구에게 요청할 때

좋은 요청 예시:

- 이 자료에서 창원시 시설별 이용률 차이를 정리해줘
- 근거 URL과 확인한 부분을 같이 남겨줘
- 결론은 중립적으로 쓰고 주민 편익 관점으로 정리해줘
- 이 답변을 산출물로 저장해줘
- 방금 보고서를 표 중심으로 다시 다듬어줘

피해야 할 요청:

- 그냥 알아서 해줘
- 대충 보고서 만들어줘
- 근거는 나중에 붙여줘

## 4. 근거와 산출물

- 지구 답변 옆에는 근거 패널이 붙습니다.
- 중요한 답변은 산출물로 저장할 수 있습니다.
- 산출물은 버전이 쌓입니다.
- 잘못된 내용이 있으면 새 버전으로 고치면 됩니다.

## 5. 운영 원칙

- 결론은 중립적으로 씁니다.
- 주민 편익과 서비스 연속성을 최상위 기준으로 둡니다.
- 선별지표와 최종판정은 분리합니다.
- 이관·통합 판단은 상대우위 근거가 있을 때만 제시합니다.
- 내부 시스템 경로나 개발 명령어는 사용자 화면에 노출하지 않습니다.

## 6. 문제가 생기면

아래 중 하나를 알려주세요.

- 접속이 안 됩니다
- 로그인은 되는데 채팅이 안 됩니다
- 답변은 오는데 근거가 없습니다
- 산출물이 저장되지 않습니다
- 파일 업로드가 실패합니다

가능하면 화면의 오류 문구도 같이 알려주세요.
